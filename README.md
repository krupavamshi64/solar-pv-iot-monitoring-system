# 🌞 Real-Time Solar PV Monitoring System Using IoT

## 📌 Overview

This project presents a **real-time performance monitoring system for solar photovoltaic (PV) panels** using Internet of Things (IoT) technology. It continuously monitors key parameters such as voltage, current, temperature, and solar irradiance to improve energy efficiency and system reliability.

The system enables **remote monitoring via a cloud platform**, helping in fault detection, performance optimization, and better energy management.

---

## 🎯 Objectives

* Monitor solar panel performance in real time
* Improve energy efficiency and power output
* Detect faults early and reduce maintenance costs
* Enable remote monitoring using IoT

---

## ⚙️ Features

* 📊 Real-time monitoring of voltage, current, temperature, and irradiance
* ☁️ Cloud-based data storage and visualization
* 📱 Remote access through web/mobile interface
* ⚠️ Fault detection and performance analysis
* 🔄 Continuous data logging
* 🔋 Energy optimization

---

## 🧠 System Architecture

The system uses sensors connected to a microcontroller (Raspberry Pi Pico W) to collect data and send it to the cloud.

### 🔹 Components Used

* Raspberry Pi Pico W
* Voltage Sensor
* Current Sensor
* Temperature Sensor
* LDR (Light Dependent Resistor)
* OLED Display
* Relay Module
* Buzzer
* Submersible Water Pump

---

## 🔌 Working Principle

1. Sensors collect real-time data from the solar panel
2. Microcontroller processes the data
3. Data is transmitted to the cloud using IoT
4. Users can view live data remotely
5. System detects faults and optimizes performance

---

## 🛠️ Tech Stack

* **Hardware:** Raspberry Pi Pico W, Sensors
* **Software:** MicroPython / Embedded C
* **Tools:** IoT Platform, Cloud Integration
* **Domain:** Embedded Systems, IoT
